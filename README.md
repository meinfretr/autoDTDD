# autoDTDD
A script to fetch the "Does the Dog Die" for a game, for use with Fossabot but can prob be used elsewhere

# How To Use
Update ${YOUR_API_KEY} with your Does the Dog Die api key, then add `$(customapi ${my-domain}/?game=$(pathencode $(game)))` as a new command to Fossabot. Adapt these instructions to your preferred chatbot tool.
