# autoDTDD
A script to fetch the "Does the Dog Die" for a game, for use with Fossabot but can prob be used elsewhere

# How To Use
Update ${MY_DTDD_API_KEY} with your Does the Dog Die api key, then add `$(customapi ${my-domain}/?game=$(pathencode $(game)))` as a new command to Fossabot. Adapt these instructions to your preferred chatbot tool. 

NOTE that this requires a Does The Dog Die API key as noted above, which is free.
